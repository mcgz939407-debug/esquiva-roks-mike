# esquiva-roks-mike
**Esquiva Roks Mike** es un juego de acción espacial para celular. Controla tu nave, dispara para destruir las rocas que aparecen y evita que se te escapen.  Cada roca destruida aumenta tu puntuación. Al destruir 50 rocas, avanzarás al siguiente nivel, donde el reto será mayor: las rocas serán más rápidas y aparecerán con mayor frecuencia.
